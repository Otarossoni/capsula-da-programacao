### Valor Zero

Quando uma variável é declarada e não possui nenhum inicializado nela, ela recebe um valor chamado *Valor Zero*. Cada tipo de dado possui um valor zero diferente. Para os tipos primitivos, os valores estão conforme o exemplo abaixo:

```go
var (
  b bool    // Output => false
  c int     // Output => 0
  d string  // Output => ""
  e float64 // Output => 0
)
```
